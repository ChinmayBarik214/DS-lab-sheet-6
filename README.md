# DS-lab-sheet-6

### Clone the repository:
```bash
git clone https://github.com/ChinmayBarik214/DS-lab-sheet-6
cd DS-lab-sheet-6
```

To Run:
```bash
javac BinaryTreeInsertion.java
java BinaryTreeInsertion
```
